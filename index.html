More Engine Cylinders Are Less Fuel Efficient

What is the relationship between the number of engine cylinders and fuel economy?

This data visualization will show that in general, more engine cylinders is associated with lower Miles Per Gallon (MPG). That is, engine cylinders are negatively correlated with MPG. We also included data points for electric vehicles. Although they don't have cylinders, they are included so the users can compare MPG among different types of engines.

How to use
  • Navigate different charts using Prev Next buttons
  • The y-axis is Highway MPG.
  • The x-axis is City MPG.
  • Circle size represents the number of Cylinders
  • The last chart allows you to select the number of Cylinders
