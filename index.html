More Engine Cylinders Are Less Fuel Efficient

What is the relationship between the number of engine cylinders and fuel economy?

This data visualization will show that more engine cylinders are associated with lower Miles Per Gallon (MPG).

How to use
  • Navigate different charts using Prev Next buttons
  • The y-axis is the number of cylinders.
  • The x-axis is logarithm scaled City MPG.
  • Circle size represents the number of Cylinders
  • The last chart allows you to select the number of Cylinders
